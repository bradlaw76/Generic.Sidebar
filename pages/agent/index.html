Generic.Sidebar Agent Docs

Your Copilot for answering questions and creating GitHub issues via MCP.

Try the Bot Open Repo Issues

What the Agent Can Do

  • Answer questions about your Generic.Sidebar project, usage patterns, and how-tos.
  • Submit GitHub issues via MCP — the agent can create (and optionally update) issues in your repository.

MCP (Model Context Protocol) enables the agent to call tools securely. For GitHub, it uses a server/tool that talks to the GitHub API with a token you control.

Quick Start

  1. Click Try the Bot and start a conversation.
  2. Ask a question: How do I embed the sidebar in Dynamics?
  3. Create an issue: Report a new issue: “Sidebar breaks when record has no start date.”

Tip: add context in your request (screenshots, steps-to-repro). The agent will include these details in the GitHub issue.

Submitting Issues via MCP

The agent supports a “create or update issue” workflow.

  • Create: Create a GitHub issue titled “Default flag flow loops” with description “Flow updates record repeatedly. Expected behavior: update once when Default=true and StartDate is null.”
  • Update: Update GitHub issue #123: add label bug and assign to @bradlaw76

Behind the scenes, the MCP GitHub tool uses your configured repository (e.g., bradlaw76/Generic.Sidebar) and token scope (repo access).

Where to Use the Agent

  • Browser: use the hosted page via the Try the Bot button above.
  • Dynamics 365 Web Resource: point an iframe to the hosted bot page:

Authentication & Permissions

  • Answering questions typically uses public project docs and agent context.
  • Creating issues requires a GitHub token with repo access configured in the MCP tool/server.
  • Your token and repo config are not stored on this static page.

Admin Notes (Optional)

  • Rotate bot: update the embed URL in /pages/copilot/index.html.
  • MCP settings: set default repo (e.g., owner/repo), labels, assignees; keep tokens in secure storage.
  • Telemetry: add GitHub Pages analytics or App Insights via a small script if desired.

FAQ

  • Can the agent update existing issues? Yes—ask it to update by number or with search terms.
  • Can it add labels or assignees? Yes, if the MCP GitHub tool is permitted to do so.
  • Does it support attachments? The agent can capture links and text; file uploads depend on your MCP implementation.
